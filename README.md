## Getting Started

1. Clone the repo:
   ```sh
   git clone https://github.com/your-username/calculator-App.git
   cd calculator-App
   ```
2. Install dependencies:
   ```sh
   yarn install
   # or
   npm install
   ```
3. Run the app:
   ```sh
   npx react-native run-android
   # or
   npx react-native run-ios
   # or (if using Expo)
   npx expo start
   ```
